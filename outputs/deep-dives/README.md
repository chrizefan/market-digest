# Deep dives (JSON-first)

Author `*.json` files with `doc_type: "deep_dive"` per `templates/schemas/deep-dive.schema.json`. Put long-form prose in `body.markdown`.

Legacy `.md` files are archived under `archive/legacy-outputs/deep-dives/`. Use `scripts/migrate_md_outputs_to_json.py` if you need to convert old markdown.

**Scaffold a new file:** `./scripts/scaffold_deep_dive.sh YYYY-MM-DD "Title Words"`
