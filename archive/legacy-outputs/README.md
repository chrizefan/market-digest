# archive/legacy-outputs/ (bulk content not in git)

Historical **markdown-era** daily trees and samples lived here. Large trees are **gitignored** so clones stay small; **source of truth is Supabase**.

To replay old disk data into the database, use the backfill scripts in [`RUNBOOK.md`](../../RUNBOOK.md) (e.g. `backfill-historical-daily-to-supabase.sh`) with a **local** copy of legacy files if you still have them.
